# intro-to-data-science
Customer Churn Prediction with Business-Oriented Model Evaluation
Project Overview

Customer churn prediction is a common machine learning problem in industries such as telecommunications, banking, and subscription services. Traditional machine learning models are usually evaluated using statistical metrics such as accuracy, precision, recall, and AUC. However, these metrics do not directly capture the business impact of prediction errors.

This project builds multiple machine learning models to predict customer churn using the Telco Customer Churn dataset and compares them using both traditional evaluation metrics and a business cost framework that accounts for the cost of false positives and false negatives.

The objective is to demonstrate that model selection should consider business impact, not just statistical performance.
